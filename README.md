# ☁️ DevOps / Platform Engineer

> From alert to resolved in one conversation

A Vienna content profile for DevOps and platform engineers. Includes quick actions for alert correlation, Terraform module generation, Kubernetes debugging, incident postmortems, cost analysis, and Dockerfile security review.

## What's included

- **Quick Actions** — Correlate alerts with metrics, generate Terraform modules, debug K8s pods, draft postmortems, analyze cloud spend, review Dockerfiles
- **Skills** — Getting started guide for Vienna profiles
- **Plugins** — Linear for incident and task tracking

## Install

Fork this profile in Vienna:

```
vienna://profile/github.com/tryvienna/profile-devops-engineer
```

Or go to **Settings → Profiles → Fork** and paste:
```
https://github.com/tryvienna/profile-devops-engineer.git
```

## Customize

After forking, this profile lives at `~/.vienna/profiles/devops-engineer/`. Edit any file to make it yours — add quick actions, swap plugins, tweak skills.

## Share

Push your customized profile to your own GitHub repo and share the `vienna://profile/...` link. Others can fork your version in one click.
